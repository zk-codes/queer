# Queer Web Directory

A curated directory of personal websites by queer folks around the world.

## Submission Guidelines

To be included in the Queer Web Directory:

- You must identify as queer
- It must be your personal site
- The site should reflect something about yourself

**Note**: Your site doesn't need to focus on queer topics or explicitly mention your identity. This is about celebrating queer folks, whatever your site.

## How To Submit Your Site

### Option 1: Submit A Pull Request

1. Fork this repository
2. Open `index.html`
3. Find the `<!-- List -->` section under "Sites"
4. Add your entry in **alphabetical order by your name**
5. Follow the exact format shown in the template below
6. Commit your changes with a clear message
7. Submit a pull request

### Option 2: Submit Via Email

Email me at [hi@zacharykai.net](mailto:hi@zacharykai.net) with:

- What you'd like to be called
- Your pronouns
- Your website URL

### Entry Template

```html
<li>
    <!-- Your Name -->
    <a href="https://yoursite.com/" rel="noopener">Your Name</a>
    <ul>
        <li><strong>Pronouns</strong>: your/pronouns</li>
    </ul>
</li>
```

## Privacy & Safety

This directory is designed with your privacy and safety in mind:

- You control how much information is displayed about you
- You can request changes or removal at any time
- Emails are never displayed publicly
- You can use a pseudonym if preferred
- You can opt out of having your pronouns displayed

## Making Changes

To update your entry or request removal:

- Submit a pull request with your changes
- Email me at [hi@zacharykai.net](mailto:hi@zacharykai.net)
- Open an issue on this repository
